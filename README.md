# nuxt-try

## Setup

```bash
# npm
npm install

or

# yarn
yarn install

```

## Start Development Server

```bash
# npm
npm run dev

or

# yarn
yarn dev
```
http://localhost:3000

## Start Product Server

```bash
# npm
npm run build

or

# yarn
yarn build
```
```bash
node .output/server/index.mjs
```
http://localhost:3000
